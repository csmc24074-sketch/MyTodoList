# 📱 My Todo List Widget App

Androidのホーム画面で完結するサクサク動作と、スマートな見た目を両立した、カレンダー＆ダークモード対応の次世代タスク管理アプリです。
Jetpack Compose と Jetpack Glance をフル活用し、スマホのシステムと美しく連動するモダンなUI/UXを実現しています。

---

## ✨ 主な機能

### 1. 📅 カレンダー連動＆スマート絞り込み
* **アプリ側:** カレンダーから「今日」「明日」「別の日」をパッと切り替えて未来・過去のタスクを自在に予定管理できます。
* **ウィジェット側:** ホーム画面には **「今日やるべきタスク」だけが自動で絞り込まれて表示** され、今集中すべきことにフォーカスできます。

### 2. 🚨 期限切れタスクの自動赤文字アラート
* タスクの設定時間を過ぎても未完了のまま残っている場合、アプリ内およびホーム画面のウィジェットの時間が、**自動的に鮮やかな赤色（太字）** に切り替わります。うっかり見落としを完全に防ぎます。

### 3. 🌓 システム連動の自動ダークモード
* スマホ本体の「ダークテーマ」の設定に一瞬で100%自動連動します。
* 夜間は目に優しいシックなダークグレーのUIに、昼間はクリーンなホワイトUIに自動着せ替えされます（アプリ・ウィジェット両対応）。

### 4. 🎨 直感的な丸型カラーバッジ（優先度）
* 従来の文字表示 `[高]` を廃止し、スマートな **「赤・オレンジ・グレー」の丸型ドットバッジ** でタスクの優先度を表現。
* 完了したタスクは自動的にグレーアウトされ、打ち消し線が入るため、進捗が一目でわかります。

### 5. ⚡ 爆速のタップレスポンス ＆ 画面レイアウト最適化
* Glance特有の遅延を限界まで排除し、ウィジェットのチェックボックスを押した瞬間に「パッ」と切り替わる市販アプリレベルの快適性を追求。
* `Window Insets` を搭載し、スマホ最上部のステータスバー（時計や電池残量表示）とアプリのタイトルが絶対に被らない自動位置調整ロジックを組んでいます。

## 🛠️ 技術スタック (Tech Stack)

* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Declarative UI)
* **Widget Framework:** Jetpack Glance (App Widget)
* **Data Storage:** Jetpack DataStore (Preferences) — データの超高速読み込み・保存
* **Architecture:** Coroutines / Flow (非同期処理・リアクティブデータストリーム)
* **Design System:** Material Design 3 (M3)

## 🚀 開発のこだわり
「毎日ホーム画面で見るものだから、無駄な文字を減らしてシンプルで美しく、かつ絶対に予定を忘れない機能性を持たせたい」というコンセプトのもと開発しました。
特にウィジェットとアプリ本体のリアルタイムなデータ同期（DataStore × Glance）の快適性と、時間の自動判定アルゴリズムにこだわっています。
